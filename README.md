# VaultX - Ultimate Crypto Security Layer

VaultX is a premium, single-page fintech/crypto landing page entirely built with **Pure HTML, CSS, and Vanilla JavaScript**. It integrates a "Dark Luxury Cyberpunk" aesthetic, merging modern web-design architecture (Glassmorphism, custom grid systems) with smooth micro-interactions.

##  Theme & Aesthetic
- **Dark Luxury Cyberpunk:** Deep void black backgrounds infused with subtle blue-black gradient mesh noise and glowing electric cyan (`#00F5D4`), deep violet (`#7B2FFF`) highlights.
- **Glassmorphism:** Premium frosted-glass effect panels implementing blur-filters and custom semi-transparent border rules.
- **Typography:** Features *Syne* for geometric, futuristic authority on headers, and *IBM Plex Mono* for monospaced, trustworthy data labels.

##  Features and Interactions
- **Responsive Architecture (Bulletproof Design):** 100% strictly responsive across all modern breakpoints using fluid typography strategies (`clamp()`), structural CSS Grids (`grid-template-columns`, `minmax()`), preventing any clipping or horizontal overflows natively.
- **Mobile Drawer Navigation:** Fully animated hamburger menu triggering a blurred, full-screen glass overlay.
- **Typewriter Effect Engine:** Vanilla JS-powered synchronized typewriter headline entry sequence.
- **Live SVG Chart Drawing:** Dynamic stroke-dasharray animations rendering asset market-charts.
- **Scroll-Triggered Observations:** Integrates the `IntersectionObserver` API for activating real-time statistical counter dials and securing lock animations strictly as they slide into the viewport.
- **Wallet Connection Maps:** Responsive absolute mapping for ecosystem wallet connections (MetaMask, Phantom, WalletConnect, etc.), falling back into static grids tightly on mobile architectures.
- **No-Code Dependencies:** Completely strictly zero dependencies (no Tailwind, React, Vue, jQuery, etc).

##  Quick Start
Simply use live server or open `index.html` in any modern web browser to view the page natively.

##  Layout & Breakdown
1. **Sticky Header:** Translucent glowing glass navbar with rapid call-to-action hooks.
2. **Hero Presentation:** Deep-tilt hover physics overlaying animated SVG mesh backgrounds.
3. **Infinite Marquee Ticker:** Ultra-smooth, infinite horizontal CSS looping ticker for simulated crypto pricing.
4. **Market Overview:** Auto-scaling price grids providing real-time illusion states.
5. **Security Visualizer:** Animated lock-and-shackle SVG showcasing multi-sig zero-knowledge traits.
6. **Wallet Integration Arena:** A simulated dynamic connector tracking all supported hardware/software wallets simultaneously.
7. **Trust & Certifications:** Staggered automated statistic climbers showcasing platform liquidity metrics.

## 📱 Accessibility & Touch Strategy
Implemented optimal `min-height` standards across tap-targets and removed parallax hover tilt equations defensively over `@media (hover: none)` breakpoints so mobile devices receive standard, ultra-smooth scaling instead.
